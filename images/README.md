# VS Code Extension Images

This directory contains images for the LDF VS Code extension.

## Required: ldf-icon.png

**Specifications:**
- Size: 128x128 pixels (minimum), 256x256 recommended
- Format: PNG with transparency
- Purpose: Extension icon shown in VS Code marketplace and sidebar

**Design suggestions:**
- Use LDF branding colors
- Simple, recognizable design that works at small sizes
- Consider including spec/checklist visual element

To publish to VS Code Marketplace, add `ldf-icon.png` to this directory.
